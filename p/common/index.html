一些常用界面

Dec 29, 2025
阅读时长: 2 分钟

  • 下面的 Win+R 可以替换成在这里输入然后回车

  • 按下 Win+X

计算机管理

  • 打开路径：

    • 右键点击“此电脑”（或“我的电脑”） -> 选择“管理”

    • 控制面板 -> 系统和安全 -> 管理工具 -> 计算机管理

  • Win+R 启动：

    1

    compmgmt.msc

设备管理器

  • 用途：查看/管理硬件设备、更新驱动、禁用设备。

  • 打开路径：

    • 右键“此电脑” → 管理 → 设备管理器
    • 控制面板 → 硬件和声音 → 设备管理器

  • Win+R 启动：

    1

    devmgmt.msc

磁盘管理

  • 用途：管理硬盘分区（扩展/压缩/格式化分区）。

  • 打开路径：

    • 右键“此电脑” → 管理 → 磁盘管理

  • Win+R 启动：

    1

    diskmgmt.msc

注册表编辑器

  • 用途：修改系统深层配置（谨慎操作）。
  • 打开路径：
    • 无直接图形入口，建议通过运行命令打开。
  • Win+R 启动： 
    1

    regedit

服务

  • 用途：管理后台服务（启动/停止/禁用）。

  • 打开路径：

    • 右键“此电脑” → 管理 → 服务和应用程序 → 服务

  • Win+R 启动：

    1

    services.msc

系统信息

  • 用途：查看详细硬件配置和系统版本。

  • 打开路径：

    • 开始菜单 → Windows 管理工具 → 系统信息

  • Win+R 启动：

    1

    msinfo32

Windows 防火墙

  • 用途：管理防火墙规则。

  • 打开路径：

    • 控制面板 → 系统和安全 → Windows Defender 防火墙

  • Win+R 启动：

    1

    firewall.cpl

事件查看器

  • 用途：查看系统/程序错误日志，排查故障。

  • 打开路径：

    • 右键“此电脑” → 管理 → 事件查看器

  • Win+R 启动：

    1

    eventvwr.msc

任务管理器

  • 用途：查看进程性能、启动项管理、结束任务。
  • 打开路径：
    • Ctrl + Shift + Esc
    • Ctrl + Alt + Del → 任务管理器
    • 右键任务栏
  • Win+R 启动： 
    1

    taskmgr

资源监视器

  • 用途：实时监控 CPU、内存、磁盘、网络使用情况。
  • 打开路径：
    • 任务管理器 → 性能选项卡 → 打开资源监视器
  • Win+R 启动： 
    1

    resmon

系统配置

  • 用途：管理启动项、服务、引导选项（如安全模式）。
  • 打开路径：
    • 任务管理器 → 文件 → 运行新任务 → 输入 msconfig
  • Win+R 启动： 
    1

    msconfig

组策略编辑器

  • 用途：高级系统设置调整（仅限 Windows 专业版及以上）。

  • 打开路径：

    • 控制面板 → 管理工具 → 组策略

  • Win+R 启动：

    1

    gpedit.msc

Windows 内存诊断工具

  • 用途：检测内存故障。
  • 打开路径：
    • 控制面板 → 管理工具 → Windows 内存诊断
  • Win+R 启动： 
    1

    mdsched

磁盘清理工具

  • 用途：清理临时文件、释放磁盘空间。
  • 打开路径：
    • 右键磁盘 → 属性 → 磁盘清理
  • Win+R 启动： 
    1

    cleanmgr

系统还原

  • 用途：恢复系统到之前的还原点。
  • 打开路径：
    • 控制面板 → 恢复 → 打开系统还原
  • Win+R 启动： 
    1

    rstrui

高级启动选项（恢复环境）

  • 用途：修复启动、安全模式、系统重置。
  • 打开路径：
    • 设置 → 更新与安全 → 恢复 → 高级启动
  • Win+R 启动： 
    1

    shutdown /r /o /f /t 0

网络连接配置

  • 用途：管理网络适配器、IP 设置。
  • 打开路径：
    • 控制面板 → 网络和共享中心 → 更改适配器设置
  • Win+R 启动： 
    1

    ncpa.cpl

电源选项

  • 用途：调整电源计划、休眠设置。
  • 打开路径：
    • 控制面板 → 硬件和声音 → 电源选项
  • Win+R 启动： 
    1

    powercfg.cpl
