重装系统

Note

重装后所有装在C盘的软件需要重新配置，环境需要重新配置，非必要不重装

重装系统后，厂商自带的软件会丢失，绝大多数都可以通过官网重新下载

方式一：首先尝试Windows系统自带的重新安装、初始化电脑功能

  • 重置： 此选项会重新安装 Windows，并让你可以选择保留或删除个人文件。 但是，应用和设置会重置为默认值。 有关详细信息，请参阅 如何重置电脑。
  • 重新安装 Windows： 此选项从头开始重新安装整个作系统，可以使用安装媒体或恢复驱动器来完成。 它更彻底，通常删除所有文件和应用。

方式二：使用预安装环境（PE）

一个微型或精简版的操作系统，通常用于系统维护和安装。PE 系统可以通过 U 盘或移动硬盘启动，不依赖于原本电脑上的操作系统。其主要功能包括、系统安装、硬盘分区、系统备份和还原、引导修复、文件管理、清除开机密码等。

使用正规品牌，容量8G以上U盘

  • 常见工具：

    • 微软官方提供了 创建 Windows 安装媒体工具

      Win11：下载 Windows 11 Win10：下载 Windows 10

    • 微PE：更精简，通常够用

      官方链接：微PE工具箱 - 下载

    • 优启通：工具更加全面

      官方链接：https://www.upe.net/ VIP版（附加一些工具和联网、外放等）：优启通EasyU_123云盘免登录下载不限速

  • U盘制作和电脑设置：

    1. 安装前转移U盘内文件，安装PE到U盘会格式化

    2. 格式化时文件格式选择 exFAT

      特性 FAT32 exFAT NTFS
      大小 最大4GB（单个分区32G） 理论上16EB（实际看设备） 理论上16EB
      **兼 全支持 Win/Mac支持好，Linux需额外驱动，老设备可能不认 主要Windows支持，Mac只读，Linux需驱动

    3. 正式开始制作前关闭安全软件，防止软件被意外拦截导致错误（软件会进行修改分区等操作）

    4. 使用启动盘时，台式机注意插到和主板直接相连的USB接口，笔记本可以随便选一个。也就是说，（避免接在延长线，拓展坞上）

    5. 进入PE

      • 插入启动盘，按开机后，台式机疯狂按Del按键进入BIOS界面，笔记本略有不同，常见型号如下：

        品牌 常用按键 (可依次尝试) 关键点与补充说明
        联想 (Lenovo) F2 或 F1 大多数消费机型用 F2，ThinkPad系列通常用F1。部分旧机型需按 Fn+F2。
        戴尔 (Dell) F2 非常统一，F2 是进入BIOS的通用键。按 F12 可进入启动菜单，再选择BIOS设置。
        惠普 (HP) F10 或 Esc F10 可直接进入BIOS。按 Esc 会先进入启动菜单，再选择进入BIOS的选项。
        华硕 (ASUS) F2 或 Del F2 更为常见。台式机或部分笔记本主板也可能用 Del 键。
        宏碁 (Acer) F2 或 Del 多数机型为 F2，部分机型可能为 Del。
        华为 (Huawei) F2 部分机型可能为 Esc 键。
        微星 (MSI) Del 或 F2 根据主板型号有所不同。
        神舟 (Hasee) Del 或 F2 多数情况下为 Del 键。

      • 也可以通过 设置 - 系统 - 恢复 - 高级重启，重启后选择疑难解答 - 高级选项 - UEFI 固件设置 进入

      1. 在设置中找到启动优先顺序选项（可能显示为Boot Menu，Boot priority等）

        还有一些电脑是打开菜单选项，再下次F12进入时就可以选择了

      • 现在的机器为UEFI引导+GUID分区表，一些老机器可能为Legacy引导+MBR分区，在手动安装系统时会用到这个知识，在选择启动项时，通常有UEFI选UEFI引导，无UEFI引导选Legacy。
        特性 UEFI + GPT (现代组合) Legacy + MBR (传统组合)
        对应关系 UEFI启动模式必须配合GPT分区表（也可兼容MBR，但不推荐）。 Legacy启动模式通常配合MBR分区表。
        硬盘容量 支持 >2TB 的大容量硬盘。 最大只支持 2TB 硬盘，超出部分无法识别使用。
        分区数量 理论上无限制（Windows下最多128个）。 最多4个主分区，或3个主分区+1个扩展分区（内含多个逻辑分区）。
        安全性 支持安全启动，可防止恶意软件在启动前加载，安全性高。 无此安全机制。
        启动速度 启动速度更快。 启动速度相对较慢。
        系统要求 Windows 8/8.1、10、11 均原生支持。 所有Windows系统都支持，但Windows 11 已强制要求UEFI+GPT。
        适用场景 2015年后出品的新电脑、新装系统的首选，尤其是使用大容量固态硬盘时。 旧电脑（约2015年前）、或需要安装旧版操作系统（如Windows 7及更早）时。

Windows 安装媒体工具

  1. 删除驱动盘所有分区

  2. 如果想要分盘（点击新建），C盘推荐100G左右，分区后还会多出MSR分区、恢复分区等分区

  3. 点击下一步后，会检测引导方式（之前选择的）和磁盘分区表格式是否匹配

微PE

  1. 分盘后在资源管理器中可以看到EFI分区和资源分区，资源分区可以当作正常U盘使用

  2. 系统安装包下载，

    • 可以通过上文提到的官网或者官方工具下载ISO镜像
    • 可以通过原版软件下载

    通常推荐使用专业版系统，下载business editions镜像，下载完成后务必进行SHA1校验（推荐使用iHasher）

    版本类别 (Edition Type) 包含版本 (Included Versions)
    Consumer Editions (消费者版) - 家庭版 (Home) - 专业版 (Professional) - 教育版 (Education)
    Business Editions (商业版) - 专业版 (Professional) - 企业版 (Enterprise)

  3. 进入PE系统后，可以直接找到iso镜像，右键 - 安装，打开setup.exe后操作同 Windows 安装媒体工具

  4. 而分区操作可以使用 pe 中的 DiskGenius 软件进行，对该软件的介绍放在下一部分

跳过联网

  • 方法一：

    1. 在联网界面按 Shift+F10 或 Shift+Fn+F10，弹出命令提示符窗口，输入：OOBE\BYPASSNRO，敲回车电脑重启
    2. 再次回到连接网络界面，点击“我没有Internet连接”即可
    3. 接着点击“继续执行受限设置”即可

  • 方法二：

    使用命令 start ms-cxh:localonly

  • 方法三：

    1. 按 Shift + F10 输入 taskmgr 启动任务管理器
    2. 在 详细信息 选项卡中找到并结束 OOBENetworkConnectionFlow.exe 或 Network Connection Flow 或 网络连接流 进程

整体界面

  • 左边列表：磁盘和分区详情

    • 以我的电脑为例：有三个磁盘，HD0、HD1、RD2，其中RD2是U盘
    • ESP、MSR 、Recovery 分区为系统需要，用户不需要管理

  • 点击磁盘或者分区会在右下板块显示详情

  • 右键磁盘出现功能列表

    Note

    这些功能一般在PE环境下操作，在主机系统下谨慎使用

    进行下列操作后，可以每次操作后或者推出软件前点击保存更改

    • 删除所有分区：用于装系统之前重置磁盘

    • 快速分区：界面如下，注意一定要选择对齐分区，不然会损失性能和寿命

    • 转换分区表类型为 GUID / MBR 格式，可以取代前面提到的命令行操作

  • 右键分区出现功能列表

    Note

    对分区的操作可以在主机系统下进行，涉及C盘的操作需要注意一下

    该工具可以帮助同学们在同磁盘其他分区有空余的情况下扩容C盘

    • 调整分区大小

      不如拆分分区方便，功能是可以将当前分区向前或向后拆分出空闲分区、新分区、或者合并到其他分区

    • 扩容分区

      主要用于扩容，点击后选择被剥削分区，出现的界面中是被剥削磁盘的情况，中间的分界线可以拖动

    • 拆分分区

      在调整分区大小的基础上，被选择分区可以拖动，前后两条线可以拖动

系统自带磁盘管理

  • 也可以进行创建分区操作，常用于安装新的硬盘后

  • 压缩卷：只能向后分

  • 拓展卷：只能吞后面的

  • 系统自带的工具限制更多（包括方向、不能充分拆分等），但比第三方工具安全很多

清理C盘

  • 主要途径

    • 清理缓存：删除C盘内文件有风险，临时文件一段时间后还会生成

    • 移动文件：如移动微信、QQ的缓存记录

  • 操作习惯

    • 先看看桌面位置在不在C盘，检查有没有大文件或文件夹，有的话可以移动到别的分区然后创建快捷方式

      Note

      不推荐移动，可能出现bug，后续不要直接放在桌面即可

      • 查看和修改位置：资源管理器左栏 - 右键桌面 - 属性 - 位置 如果左栏没有：则在桌面创建一个新文件，右键查看位置，如 C:\Users\aoruo\Desktop，访问C:\Users\Desktop，既可以看到桌面图标

    • 后续尽量避免在C盘装软件

  • 清理缓存：

    1. Win+r 打开启动，输入 %temp%，跳转临时文件文件夹，将文件夹内内容删除 如果开启了回收站，可以使用 Ctrl+Del 进行永久删除

      Caution

      主要不要删除TEMP文件夹

      Note

      碰到某某文件正在被使用提醒，选择跳过即可 如果长时间卡住，可以点击取消或者直接关闭进度窗口，之后重新删除（Windows在清理大量小文件时容易出现该情况）

    2. 清理临时文件

      • win11：右键磁盘 - 属性 - 常规 - 详细信息 - 临时文件，默认选项清理即可，如果不需要保留回收站，则勾选

      • win10：右键磁盘 - 属性 - 常规 - 磁盘清理

    3. 关闭休眠，笔记本有睡眠选项

      • Win+s，搜索 cmd，管理员模式打开

      • 首先查看是否开启 powercfg /a

      • 关闭 powercfg.exe /hibernate off，可以缩写为 powercfg /h off

      • 开启 powercfg.exe /hibernate on，可以缩写为 powercfg /h on

  • 使用火绒、360等软件的垃圾清理

  • DISM++

    该工具可以用于清理C盘

    github链接：Chuyu-Team/Dism-Multi-language: Dism++ Multi-language Support & BUG Report

    Note

    如果回收站内容还有用，或者占用空间不大，取消勾选
